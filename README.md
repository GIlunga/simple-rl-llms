# Simple RL for LLMs

The goal of this project is to create a lightweight LLM RL fine-tuning codebase that can be used to experiment with different algorithms and compare with existing papers at a smaller scale (<8B parameters). All experiment reports will be kept in this repo and on gilunga.github.io.

Current goals:
- Get a simple GRPO baseline training a 0.5B model on one environment with a TRL baseline
- Write about it
